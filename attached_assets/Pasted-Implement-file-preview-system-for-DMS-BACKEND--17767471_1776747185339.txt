Implement file preview system for DMS.

----------------------------------------
BACKEND
----------------------------------------

Create API:

GET /api/dms/file-preview?id={fileId}

----------------------------------------
LOGIC
----------------------------------------

1. Fetch file:

const file = await prisma.dmsDocument.findUnique({ where: { id } })

2. Permission check

3. Get file stream from SharePoint

4. Set headers:

res.setHeader("Content-Type", file.mime_type)

----------------------------------------
IMPORTANT
----------------------------------------

✔ DO NOT force download
✔ Stream file inline

----------------------------------------
FRONTEND
----------------------------------------

----------------------------------------
SUPPORTED TYPES
----------------------------------------

Preview inline if:

✔ PDF → application/pdf
✔ Images → image/jpeg, image/png, image/webp

Else → fallback

----------------------------------------

----------------------------------------
