Upgrade Branding system to full application-level control.

----------------------------------------
STEP 1 — UPDATE SCHEMA
----------------------------------------

Add new fields in Branding model:

model Branding {
  id               String   @id @default(uuid())
  company_id       String   @unique

  app_name         String?     // "Compliance & AMC"
  browser_title    String?

  logo_base64      String?

  login_banner     String?     // base64 image
  login_footer     String?     // text below button
  login_bg         String?     // color or image

  primary_color    String?
  secondary_color  String?

  theme_mode       String   @default("light")

  created_at       DateTime @default(now())
  updated_at       DateTime @updatedAt

  company Company @relation(fields: [company_id], references: [id])
}

----------------------------------------
RUN
----------------------------------------

npx prisma migrate dev
