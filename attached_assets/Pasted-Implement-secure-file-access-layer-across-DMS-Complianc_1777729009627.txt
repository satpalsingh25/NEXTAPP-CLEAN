Implement secure file access layer across DMS, Compliance, AMC.

----------------------------------------
STEP 1 — CREATE SECURE DOWNLOAD API
----------------------------------------

Create:

/app/api/files/download/[id]/route.ts

----------------------------------------
CODE REQUIREMENTS
----------------------------------------

✔ Require authentication
✔ Validate file belongs to req.user.company_id
✔ Validate file exists
✔ Fetch file from SharePoint using internal path
✔ Return file buffer
✔ Force attachment download
✔ Log audit event

----------------------------------------
RESPONSE HEADERS
----------------------------------------

Content-Type: file.mime_type
Content-Disposition: attachment; filename="file.name"
