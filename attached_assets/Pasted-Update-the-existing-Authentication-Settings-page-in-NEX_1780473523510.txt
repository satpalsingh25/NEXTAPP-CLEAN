====================================================
1. AZURE AD GUIDE
====================================================

Title:
Azure AD App Registration Setup

Show steps:

1. Go to:
Azure Portal → Azure Active Directory → App registrations → New registration

2. Set:
- App name
- Supported account types
- Redirect URI

3. Redirect URI:

https://YOUR_DOMAIN/api/auth/azure/callback

4. Copy:
- Application (client) ID
- Directory (tenant) ID

5. Go to:
Certificates & secrets → New client secret

6. Copy secret value

7. Add Microsoft Graph delegated permissions:
- openid
- profile
- email
- User.Read

8. For sync also add application permissions:
- User.Read.All
- Group.Read.All

9. Click:
Grant admin consent

====================================================
2. GOOGLE WORKSPACE GUIDE
====================================================

Title:
Google Workspace / Google Login Setup

Explain:
Google authentication works through OIDC Provider.

Steps:

1. Open:
Google Cloud Console

2. Create:
New Project

3. Enable:
OAuth APIs / Google Identity Services

4. Configure:
OAuth Consent Screen

5. Create:
OAuth 2.0 Client ID

Application Type:
Web Application

6. Add Redirect URI:

https://YOUR_DOMAIN/api/auth/oidc/callback

7. Copy:
- Client ID
- Client Secret

8. Add OIDC Provider in NEXTAPP-CLEAN

9. Use:

Issuer URL:
https://accounts.google.com

Discovery URL:
https://accounts.google.com/.well-known/openid-configuration

Scopes:
openid profile email

====================================================
3. LDAP / ACTIVE DIRECTORY GUIDE
====================================================

Title:
LDAP / Active Directory Setup

Steps:

4. Fill:
- LDAP URL

5. Enable:
LDAP Login

Import Users
