STEP 6 — THEME TOGGLE (GLOBAL)
----------------------------------------

LOCATION:

Top-right corner near Logout button

----------------------------------------
UI
----------------------------------------

Icon toggle:

🌞 Light | 🌙 Dark

----------------------------------------
STATE
----------------------------------------

const [theme, setTheme]

----------------------------------------
LOGIC
----------------------------------------

onToggle:

if (theme === "light") {
  setTheme("dark")
  document.body.classList.add("dark")
} else {
  setTheme("light")
  document.body.classList.remove("dark")
}

----------------------------------------
PERSISTENCE
----------------------------------------

Use localStorage:

localStorage.setItem("theme", theme)

on load:

const savedTheme = localStorage.getItem("theme")

----------------------------------------
PRIORITY RULE
----------------------------------------

User preference overrides branding default

----------------------------------------
STEP 7 — INITIAL LOAD
----------------------------------------

On app start:

if (userPreference exists):
  apply that
else:
  apply branding.theme_mode
